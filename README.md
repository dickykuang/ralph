# ralph

Ralph command/skill prompts for Claude and Codex.

## Layout

- `claude/`: Claude slash command files (`.md`)
- `codex/`: Codex skill directories (each with `SKILL.md`)

## Install

```bash
./install.sh claude
./install.sh codex
```

- `claude` installs to `~/.claude/commands`
- `codex` installs to `~/.codex/skills`

## Uninstall

```bash
./uninstall.sh claude
./uninstall.sh codex
```
